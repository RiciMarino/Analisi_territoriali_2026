La sezione **Attività regionali** usa esclusivamente le righe del foglio `BASE DATI`
nelle quali la colonna `Tipo` contiene il valore `Regionali`.

Le attività sono raggruppate per disciplina, categoria e regione, con ordinamento
decrescente in base al numero di squadre.
