# Attività di squadra 2026

Sito statico autonomo per GitHub Pages.

## File da caricare nella nuova repository

- `index.html`
- `logo_csi.jpg`
- `README.md`

La dashboard è contenuta direttamente in `index.html`: non esistono reindirizzamenti o file provenienti da altri progetti.

## Struttura del sito

- Riepilogo attività
- Soddisfa C.U.
- Non soddisfa C.U.
- Mostra dati insieme

Le righe con la dicitura `Squadre insufficienti` nella colonna NOTE sono inserite nel tab `Non soddisfa C.U.`.

## Pubblicazione con GitHub Pages

1. Creare una nuova repository pubblica.
2. Caricare nella root i tre file del pacchetto.
3. Aprire `Settings` → `Pages`.
4. In `Build and deployment` scegliere `Deploy from a branch`.
5. Selezionare il branch `main` e la cartella `/ (root)`.
